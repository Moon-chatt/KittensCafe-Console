# KittensCafe-Console

Програмне забезпечення на базі платформи .NET для автоматизації розрахунку та керування фінансовими чеками в закладах громадського харчування. Проєкт реалізовано у вигляді консольного додатка (CLI) з чітким розділенням відповідальності між бізнес-логікою та представленням.

---

## Основні функціональні можливості

* **Швидке керування в інтерфейсі CLI:** Вибір пунктів меню реалізовано через пряме перехоплення натискання клавіш (Console.ReadKey(true)). Користувачеві достатньо натиснути цифру від 0 до 7, без необхідності підтверджувати вибір клавішею Enter.
* **Обмеження обсягу чека:** Логіка програми використовує статичний масив фіксованого розміру (максимум 5 елементів), що дозволяє ефективно контролювати споживання пам'яті.
* **Гнучка система чайових:** Додаток підтримує нарахування чайових двома способами — у вигляді відсотка від суми замовлення або як фіксовану грошову суму.
* **Форматований вивід фінансів:** Фінальний чек відображається у вигляді структурованої таблиці з вирівнюванням по колонках. Розрахунок податку GST (5%) та загальної суми проводиться автоматично в режимі реального часу.
* **Текстовий імпорт/експорт:** Збереження поточного замовлення у файл та зчитування накопичених даних для продовження роботи.

---

## Архітектура та структура проєкту

Кодова база розділена на три фізичні файли відповідно до їхніх функціональних обов'язків:

* `ConsoleUI.cs` — клас представлення. Відповідає за відображення графічного ASCII-меню, зчитування команд користувача та форматований вивід інформації на екран.
* `OrderManager.cs` — клас бізнес-логіки. Відповідає за збереження стану замовлення, математичні розрахунки, валідацію вхідних параметрів та запис/читання файлів. Також містить у собі допоміжний клас-модель MenuItem (опис та ціна позиції).
* `Program.cs` — точка входу в додаток. Ініціалізує об'єкти менеджера та інтерфейсу, запускає головний цикл програми.
